# mochi
</div>

A Rusty concatenative programming language

## Installation

### From registry

```console
$ cargo install mochi-lang
```

### From source

```console
$ git clone https://github.com/lebakasable/mochi
$ cargo install --path .
```

## Quick Start

```console
$ cargo run -- examples/fizzbuzz.moc -r
```
